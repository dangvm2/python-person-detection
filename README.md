

# How to run

1. Go to path: **python-person-detection/**
2. Install all packages (from requirements.txt) by Command line: 
    ```sh
    pip3 install -r requirements.txt
    ```
3. Start web by Command line: 
    ```sh
    python3 app.py
    ```
4. Enter URL in browser:
    ```sh
    http://127.0.0.1:5000/
    ```
